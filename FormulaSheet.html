Differentiation and Integration Formula Sheet

Differentiation Formulas

1. Constant Rule:
d/dx(c) = 0
2. Power Rule:
d/dx(x^n) = n * x^(n-1)
3. Sum/Difference Rule:
d/dx(f(x) ± g(x)) = d/dx(f(x)) ± d/dx(g(x))
4. Product Rule:
d/dx(f(x) * g(x)) = f'(x) * g(x) + f(x) * g'(x)
5. Quotient Rule:
d/dx(f(x) / g(x)) = [f'(x) * g(x) - f(x) * g'(x)] / (g(x))^2
6. Chain Rule:
d/dx(f(g(x))) = f'(g(x)) * g'(x)
7. Exponential Functions:
d/dx(e^x) = e^x
d/dx(a^x) = a^x * ln(a)
8. Logarithmic Functions:
d/dx(ln(x)) = 1/x
d/dx(log_a(x)) = 1/(x * ln(a))
9. Trigonometric Functions:
d/dx(sin(x)) = cos(x)
d/dx(cos(x)) = -sin(x)
d/dx(tan(x)) = sec^2(x)
d/dx(cot(x)) = -csc^2(x)
d/dx(sec(x)) = sec(x) * tan(x)
d/dx(csc(x)) = -csc(x) * cot(x)
10. Inverse Trigonometric Functions:
d/dx(sin^-1(x)) = 1/√(1 - x^2)
d/dx(cos^-1(x)) = -1/√(1 - x^2)
d/dx(tan^-1(x)) = 1/(1 + x^2)

Integration Formulas

1. Power Rule (For Integration):
∫x^n dx = x^(n+1)/(n+1) + C
2. Sum/Difference Rule (For Integration):
∫(f(x) ± g(x)) dx = ∫f(x) dx ± ∫g(x) dx
3. Exponential Functions:
∫e^x dx = e^x + C
∫a^x dx = a^x / ln(a) + C
4. Logarithmic Functions:
∫1/x dx = ln|x| + C
5. Trigonometric Functions:
∫sin(x) dx = -cos(x) + C
∫cos(x) dx = sin(x) + C
∫tan(x) dx = ln|sec(x)| + C
∫cot(x) dx = ln|sin(x)| + C
∫sec(x) dx = ln|sec(x) + tan(x)| + C
∫csc(x) dx = -ln|csc(x) + cot(x)| + C
6. Inverse Trigonometric Functions:
∫1/√(1 - x^2) dx = sin^-1(x) + C
∫1/(1 + x^2) dx = tan^-1(x) + C
7. Substitution (Change of Variable):
∫f(g(x))g'(x) dx = ∫f(u) du
8. Integration by Parts:
∫u dv = uv - ∫v du
